Digest::Class.digest(string, *parameters) → hash_string

Returns the hash value of a given string. This is equivalent to Digest::Instance#new.digest(string), where extra parameters, if any, are passed through to the constructor and the string is passed to Digest::Instance#digest.

Source: show 

static VALUE
rb_digest_class_s_digest(int argc, VALUE *argv, VALUE klass)
{
    VALUE str;
    volatile VALUE obj;

    if (argc < 1) {
        rb_raise(rb_eArgError, "no data given");
    }

    str = *argv++;
    argc--;

    StringValue(str);

    obj = rb_obj_alloc(klass);
    rb_obj_call_init(obj, argc, argv);

    return rb_funcall(obj, id_digest, 1, str);
}
Digest::Class.hexdigest(string[, ...]) → hash_string

Returns the hex-encoded hash value of a given string. This is almost equivalent to Digest.hexencode(Digest::Instance#new.digest(string)).
